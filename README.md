# django_test


### Use Python
#### Install
* Install `Python v3.9`
* Run `pip install -r requirements.txt` to install requirements

#### Finish
* Run `python .\manage.py runserver`
* Starting server at http://localhost:8000/



### Use Docker
* `docker-compose up`
* Starting server at http://localhost:8000/
